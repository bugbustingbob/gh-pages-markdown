# Git Cheatsheet
Some handy notes to help me remember `git` commands.

## Basic
* `git init`
* `git add .`
* `git commit -m "..."`
* `git log`
* `git diff`
* `git show`

## Remote
* `git remote add [alias] ...`
* `git fetch [alias]`
* `git merge [alias]/[branch]`
* `git push [alias] [branch]`
* `git pull`
